Projects

Robotic arm playing beer pong autonomously

As part of a BSc project, I worked with 3 other students to build software for a robotic arm that plays beer pong autonomously. We fine-tuned YOLO to detect cups and the ball, then used a C library to control the arm based on overhead camera input. We also implemented aim auto-adjustment for missed shots.

3D renderer in C++ with zero dependencies

The renderer loads an .obj file, with or without texture mapping, and paints a scene into .tga file. It supports basic lighting and camera movement, z-buffering, Gouraud and Phong shading models.

Java Tetris with autoplay

Java tetris setup for gaming and autoplay simulations. You can play yourself or run an autoplay visualization. Autoplay parameters were optimized using a genetic algorithm.

3D image-stitching tool

A 3D image-stitching tool that aligns two overlapping photos (e.g., from a panorama) by estimating their homography transform with Robust Regression and warping one of the images onto the other.
